Non-cyclic patterns

The cyclic patterns (critertion 1-4) are contained in the path patterns (criterion 1,3,4). Thus the non-cyclic patterns are simply the path patterns with the cyclic patterns taken away. I.e. to find the number of non-cyclic patterns of length L, we take the difference between the number of cyclic patterns of length L and the number of path patterns of length L. For L=24 we get

58948-10792=48156
