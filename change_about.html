About change ringing patterns

Let n be a positive integer. A change ringing pattern for n bells can be represented by a sequence of permutations of the set {1,2,...,n-1,n} that satisfies the following criteria.

  1. The sequence must start with the permutation 
     (1,2,...,n-1,n)
  2. The sequence must end with the starting permutation.
  3. No permutation can be repeated except for the starting permutation which can be repeated at most once at the end of the pattern to accommodate criterion 2.
  4. The position of each bell from one permutation to the next can stay the same or move by at most one place.

An optional and very common criterion is

  1. The sequence must contain all permutations.

We define a full pattern as a pattern that satisfies all criteria.

We define a short pattern as a pattern that satisfies all criteria except criterion 5.

We define the length of a change ringing pattern as the number of unique permutations in the pattern.

A full pattern contains all permutations. There are factorial(n) permutations of n bells, so the length of a full pattern is always factorial(n).

A short pattern does not contain all permutations, hence it has a length less than factorial(n).

We will list both full and short patterns on this page. We will categorize patterns by their length. There is only one pattern of length 1 for any n, namely the pattern consisting solely the starting permutation. There are no patterns of length 0 since all patterns must at least have the starting permutation (criterion 1).

What about patterns for 0 bells? Remember that n is a positive integer, hence per definition there are no change ringing pattern for 0 bells. What is the reason for this? Imagine a pattern for 0 bells. Since there are no bells to permute, there are no patterns of any length.
