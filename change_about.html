About change ringing patterns

Let n be a positive integer. A change ringing pattern for n bells can be represented by a sequence of permutations of the set {1,2,...,n-1,n} that satisfies the following criteria.

  1. The position of each bell from one permutation to the next can stay the same or move by at most one place.
  2. No permutation can be repeated except for the starting permutation which can be repeated at most once at the end of the pattern to accommodate criterion 4.

Optional criteria

  1. The sequence must start with the permutation (1,2,...,n-1,n)
  2. The sequence must end with the same permutation that it started with.
  3. The sequence must contain all permutations.

We define a full pattern as a pattern that satisfies all five criteria.

We define a short pattern as a pattern that satisfies criteria 1-4, but does not satisfy criterion 5.

We define the length of a change ringing pattern as the number of unique permutations in the pattern.

A full pattern contains all permutations. There are factorial(n) permutations of n bells, so the length of a full pattern is always factorial(n).

A short pattern does not contain all permutations, hence it has a length less than factorial(n).

There is only one pattern of length 1, namely the pattern consisting solely the starting permutation. There are no patterns of length 0 since all patterns must at least have the starting permutation (criterion 1).

We define a cyclic pattern as a pattern that satisfies criteria 1-4. "Cyclic" because such patterns start and end with the same permutation.

We define a path pattern as a pattern that satisfies criteria 1-3. "Path" because such patterns are not necessarily cyclic, although every cyclic pattern is also a path pattern.

We define a non-cyclic pattern as a pattern that satisfies criteria 1-3, but does not satisfy criterion 4. I.e. a non-cyclic pattern is a path pattern that is not cyclic.
