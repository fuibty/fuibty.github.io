About change ringing patterns

A change ringing pattern with n bells is a sequence of permutations of the set {1,2,...,n-1,n} that satisfies the following criteria.

  1. The sequence must start with the permutation 
     1,2,...,(n-1),n
  2. The sequence must end with the same permutation that it started with.
  3. No permutation can be repeated except for the starting permutation which must be repeated once in accordance with criterion 2.
  4. The position of each bell from one permutation to the next can stay the same or move by at most one place.

An optional criterion is

  1. The sequence must contain all permutations, i.e. pattern has length factorial(n).

We define the length of a change ringing pattern as the number of unique permutations in the pattern.

A change ringing pattern that satisfy criterion 5 is called a full pattern.

A change ringing pattern that does not satisfy criterion 5 is called a short pattern.

We will list both full and short patterns. We will categorize short patterns by their length. Short patterns have length < factorial(n). This gives n-1 categories of short patterns. We will not bother listing patterns of length 0 (empty) and length 1 (only the starting permutation).
